# Zen Browser for Android (Unofficial) 🚀

> [!IMPORTANT]
> This is an **unofficial** port of Zen Browser for Android. It is a **hobby project** created for personal use and to explore the limits of mobile browsing.
> 
> This project was developed using **Antigravity** and other **vibe coding** tools, emphasizing a rapid, AI-assisted development workflow.

Zen Browser for Android brings the sleek, desktop-inspired browsing experience to your mobile device. Based on the powerful GeckoView engine, it aims to provide a premium, customizable, and "desktop-first" feel on Android.

## ✨ Features
- **Desktop Experience:** Vertical tabs, advanced sidebar, and a layout optimized for efficiency (UI logic adapted from Zen Desktop).
- **Privacy First:** Built-in tracking protection and privacy-focused defaults.
- **Extensible:** Support for a wide range of Firefox extensions.
- **Modern Architecture:** Built on Android 14 (SDK 34) for maximum stability and performance.

## 📥 Installation

Releases are published as `.apk` files. Currently, you can find them in the [Releases](https://github.com/ApoBen/Zen-Android/releases) section.

1. **Determine your version**: 
   - **arm64-v8a**: Most modern 64-bit phones (Highly Recommended).
   - **armeabi-v7a**: Older 32-bit phones.
2. [**Download the APK**](https://github.com/ApoBen/Zen-Android/releases).
3. **Install and Enjoy**. You may need to enable "Install from Unknown Sources".

## 🔨 Building

1. Ensure you have the Android SDK and Java 17+.
2. Clone the repository:
   ```sh
   git clone https://github.com/ApoBen/Zen-Android.git
   ```
3. Run the build:
   ```sh
   ./gradlew assembleDebug
   ```
